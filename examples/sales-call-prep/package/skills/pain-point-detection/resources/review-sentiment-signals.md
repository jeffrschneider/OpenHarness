# Review Sentiment Signal Analysis

## Glassdoor Analysis

### Where to Look
- "Cons" section (direct complaints)
- "Advice to Management" (systemic issues)
- Reviews from relevant departments
- Recent reviews (last 6-12 months)
- Verified employee reviews

### Common Pain Point Patterns

| Phrase Cluster | Underlying Pain | Sales Angle |
|----------------|-----------------|-------------|
| "Growing pains", "scaling too fast", "chaos" | Systems can't keep up | Scale-ready solution |
| "Too many tools", "fragmented", "disconnected" | Tool sprawl | Consolidation |
| "Manual processes", "tedious", "repetitive" | Automation gap | Workflow automation |
| "Communication issues", "silos", "politics" | Collaboration breakdown | Unified platform |
| "Old technology", "outdated systems", "legacy" | Technical debt | Modernization |
| "Burnout", "overworked", "understaffed" | Capacity issues | Efficiency gains |
| "No clear process", "ad hoc", "inconsistent" | Process maturity | Best practices |

### Department-Specific Signals

#### Engineering Reviews
- "Tech debt" = Modernization opportunity
- "Deploy process" = DevOps needs
- "Testing" = Quality tooling

#### Sales/Marketing Reviews
- "CRM issues" = Sales tech need
- "Lead quality" = Marketing tech need
- "Reporting" = Analytics need

#### Operations Reviews
- "Manual work" = Automation
- "Data entry" = Integration
- "Approval bottlenecks" = Workflow

### Rating Trends

| Trend | Interpretation |
|-------|----------------|
| Declining ratings | Problems worsening |
| Stable low ratings | Chronic issues |
| Improving ratings | Leadership action |
| Volatile ratings | Inconsistent experience |

## G2/Capterra Analysis (B2B Products)

### Competitor Product Reviews

If they use a competitor, check reviews for:
- Feature complaints (gaps you fill)
- Support issues (service differentiation)
- Price complaints (value positioning)
- Integration problems (compatibility angle)

### Category Research

Search their industry + pain point:
- "[Industry] + operations software reviews"
- "[Industry] + workflow automation reviews"

## Social Media Sentiment

### LinkedIn
- Company page comments
- Employee posts (public complaints rare but telling)
- Leadership posts and engagement

### Twitter/X
- @company mentions
- Employee complaints (search "[company] + frustrated")
- Customer complaints

### Reddit
- Industry subreddits
- Company-specific subreddits (for large companies)
- Anonymous employee posts

## Analysis Framework

For each review source, capture:

```
Source: [Glassdoor/G2/etc.]
Date Range: [Reviews from when]
Sample Size: [Number of reviews]
Key Themes:
  1. [Theme] - [Frequency: High/Med/Low]
  2. [Theme] - [Frequency]
Pain Point Summary: [1-2 sentences]
Relevant Quote: "[Direct quote from review]"
Sales Angle: [How to reference in conversation]
```

## Caution Notes

- Don't quote negative reviews directly to prospects
- Use themes, not specific complaints
- Frame as "challenges I've heard about in your industry"
- Reviews may be outdated or unrepresentative
