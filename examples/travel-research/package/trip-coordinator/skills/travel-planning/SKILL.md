# Travel Planning Skill

Core knowledge for planning and formatting travel itineraries.

## Capabilities

- Structure multi-day itineraries with logical flow
- Calculate and track budgets across categories
- Optimize travel dates for pricing and weather
- Format outputs for easy reading and sharing

## Resources

- `itinerary-template.md` - Standard format for day-by-day plans
- `packing-checklist-template.md` - Base packing list by trip type

## Scripts

- `budget-calculator.py` - Track spending across categories
- `date-optimizer.py` - Find optimal travel windows
- `itinerary-formatter.py` - Generate formatted itineraries
