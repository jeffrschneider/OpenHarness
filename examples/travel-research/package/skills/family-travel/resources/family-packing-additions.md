# Family Travel Packing Additions

Items to add beyond standard packing list when traveling with children.

## All Ages - Essentials

### Documents
- [ ] Child's passport
- [ ] Birth certificate (for domestic travel)
- [ ] Custody documentation (if applicable)
- [ ] Insurance cards
- [ ] Pediatrician contact info
- [ ] Allergy/medical information cards

### Health & Safety
- [ ] Children's medications (fever reducer, allergy meds)
- [ ] Band-aids and antiseptic
- [ ] Thermometer
- [ ] Sunscreen (kid-friendly SPF 50+)
- [ ] Insect repellent (kid-safe)
- [ ] Hand sanitizer

### Comfort Items
- [ ] Favorite stuffed animal/blanket
- [ ] Familiar snacks
- [ ] Comfort items for sleep

## Babies & Toddlers (0-3)

### Gear
- [ ] Car seat (or plan for rental)
- [ ] Stroller (lightweight travel version)
- [ ] Baby carrier/wrap
- [ ] Portable crib/pack-n-play (check hotel availability)
- [ ] High chair clip-on (optional)

### Feeding
- [ ] Formula/bottles (enough for delays)
- [ ] Bottle brush
- [ ] Bibs
- [ ] Sippy cups
- [ ] Baby food/pouches
- [ ] Spoons
- [ ] Nursing cover (if applicable)

### Diapering
- [ ] Diapers (enough for travel + delays)
- [ ] Wipes (travel packs)
- [ ] Diaper cream
- [ ] Changing pad
- [ ] Disposable bags for dirty diapers
- [ ] Extra outfit per day (accidents happen)

### Sleep
- [ ] White noise machine/app
- [ ] Sleep sack
- [ ] Blackout curtains (portable)
- [ ] Night light

### Entertainment
- [ ] Board books
- [ ] Soft toys
- [ ] Teething toys
- [ ] Tablet with downloaded shows (for emergencies)

## Preschoolers (3-5)

### Activities
- [ ] Coloring books + crayons
- [ ] Sticker books
- [ ] Play-doh (small containers)
- [ ] Favorite small toys
- [ ] Tablet with games/shows
- [ ] Headphones (kid-sized)
- [ ] Books

### Clothing Additions
- [ ] Extra underwear (potty accidents)
- [ ] Pull-ups for travel/night (if needed)
- [ ] Comfortable shoes with velcro
- [ ] Rain jacket
- [ ] Hat

### Comfort
- [ ] Special blanket
- [ ] Stuffed animal
- [ ] Nightlight
- [ ] Familiar cup/plate

## School Age (6-12)

### Activities
- [ ] Books/e-reader
- [ ] Travel games
- [ ] Journal + colored pencils
- [ ] Tablet/device + charger
- [ ] Headphones
- [ ] Camera (disposable or kid-friendly)
- [ ] Card games

### Clothing Additions
- [ ] Swimsuit (if applicable)
- [ ] Light layers
- [ ] Comfortable walking shoes (broken in!)
- [ ] Flip flops for pool/beach

### Independence Items
- [ ] Small backpack (day pack)
- [ ] Water bottle
- [ ] Watch
- [ ] Small amount of spending money

## Teens (13+)

### Electronics
- [ ] Phone + charger
- [ ] Portable battery
- [ ] Headphones
- [ ] Adapter for destination

### Activities
- [ ] Books
- [ ] Journal
- [ ] Music
- [ ] Downloaded content

### Personal Items
- [ ] Toiletries (they care about brands now)
- [ ] Hair styling tools
- [ ] Skincare
- [ ] Spending money

## Airplane Essentials

### Carry-On Must-Haves
- [ ] Snacks (more than you think)
- [ ] Entertainment for each child
- [ ] Change of clothes (for everyone)
- [ ] Diapers/wipes (babies)
- [ ] Comfort items
- [ ] Medications
- [ ] Empty water bottles (fill after security)
- [ ] Gum or sucking candy (ear pressure)

### Tips
- New toys/activities revealed hourly
- Download content before flying
- Pack a "surprise bag" with wrapped small items
- Window seats for kids = entertainment
- Aisle for quick bathroom access
