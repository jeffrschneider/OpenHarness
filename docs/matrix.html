Harness Profiles

Anthropic Agent SDK

openharness-anthropic-agent
Strengths
  • Direct Anthropic Messages API access
  • Native streaming with async generators
  • Tool use with automatic agentic loop
  • Extended thinking support
  • In-memory conversation history
Limitations
  • Single provider (Anthropic only)
  • No persistent memory
  • No MCP support
  • No file operations

Letta

openharness-letta
Strengths
  • Memory-first architecture
  • Full agent lifecycle (CRUD)
  • Memory blocks with semantic search
  • Archival memory persistence
  • Cloud API support
Limitations
  • No subagent delegation
  • Limited file operations
  • No hooks system

Goose

openharness-goose
Strengths
  • MCP-first architecture
  • Multi-model support (25+ providers)
  • Local CLI + Cloud REST API modes
  • Working directory context
  • Native skills via MCP extensions
Limitations
  • Stateless execution (no sessions)
  • Tools via config only
  • No persistent memory

LangChain Deep Agent

openharness-deepagent
Strengths
  • Built-in planning/todos
  • Native subagent delegation
  • Flexible filesystem backends
  • Multi-model via LangChain
  • File operations (read, write, glob, grep)
Limitations
  • No native skills API
  • Stateless by default
  • No hooks system
